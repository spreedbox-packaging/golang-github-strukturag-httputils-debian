httputils
======================

[Go](http://golang.org) support library for HTTP servers.

## Installation

```bash
$ go get github.com/strukturag/httputils
```

## Run unit tests

```bash
$ go test -v
```

## Contributing

1. "Fork".
2. Make a feature branch.
3. Make changes.
4. Do your commits (run ``go fmt`` before commit).
5. Send "pull request".
